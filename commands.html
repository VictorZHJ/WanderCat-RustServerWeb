指令Commands


Friend

好友： /friend

新增好友： /friend + Name

刪除好友： /friend - Name

確認好友狀態： /friend list Name

好友PK： /ff on | /ff off


TP

傳送到某玩家身邊： /tpr Name

接受玩家的傳送： /tpa

取消傳送 | 拒絕傳送： /tpc

交易村： /town(手上請勿持有任何東西)


Home

傳送到家裡： /home Home

設置一個家： /home add Home

已設置的家： /home list

刪除家： /home remove Home


Tool

彩色ID： /colours

可領取的禮包： /kit

設定自動關門： /ad 10 (5-30秒)

查詢線上玩家： /players | /who

設定武器準心： F5

移除工具： /remove | /remove 10


Clan

/clan - 顯示有關您家族的相關訊息

/clan help - 顯示有關家族系統的相關指令

/c 訊息... - 家族對話

/clan create \"標籤\" \"描述\" - 創建一個您所擁有的新家族

/clan join \"標籤\" - 申請加入家族

/clan leave - 離開您目前的家族


家族管理員 指令：

/clan invite \"Player name\" - 邀請玩家進入家族

/clan kick \"Player name\" - 從家族踢一個成員


家族擁有者 指令：

/clan promote \"Name\" - 將成員提升為管理員

/clan demote \"Name\" - 將管理員降級為成員

/clan disband forever - 解散你的家族（無法取消，注意！！）"
